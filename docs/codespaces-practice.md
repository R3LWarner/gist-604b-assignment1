## Understanding Git Workflow
Explain in 2–3 sentences the difference between:
- Local workspace (Codespace)
- Remote repository (GitHub.com)
The Local workspace is a virtual environment to develop and edit code attached to the remote repository.
The Remote repository is a directery of files documenting and detailing the project.
The Local workspace makes a copy of the Remote repository to allow editations without disrupting the repository and must be commited to update the repository.
